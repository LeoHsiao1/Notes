# Linux

：一个流行的操作系统，全称为 GNU/Linux 。
- 发音为 `/ˈlɪnəks/` 。

## 版本

- [官网](https://www.kernel.org/)
- 目前 Linus Torvalds 依然主管开发 Linux 内核，源代码托管在 [GitHUb](https://github.com/torvalds/linux) 上。
- Linux 内核（kernel）的版本号命名规则为 ` 主版本.次版本.发布版本-修订版本 ` ，例如 `3.10.0-11` 。
  - 2011 年发布 v3.0 版本。
  - 2015 年发布 v4.0 版本。
  - 2019 年发布 v5.0 版本。
  - 2022 年发布 v6.0 版本。
